# Tic Tac Toe Web Application


## Description
This is a simple **Tic Tac Toe** web application, geaturing 2 game mode available. 
Players can choose to play either **vs Player** or **vs Bot**

## Tech Stack Used
- HTML
- CSS
- JavaScript

## Features
- 2-player gameplay (vs Player)
- Gameplay against a bot (vs Bot)
- Win and draw detection
- Simple and Rresponsive user interface
